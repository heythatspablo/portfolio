The Tech Stack

Before we write a single line of code (or ask AI to write it for us), let's define our tools. We want speed, performance, and aesthetics.

  • Frontend: React + Vite (Fast, standard).

  • Styling: Tailwind CSS (For that specific "Cyber/LED" look).

  • Backend: Supabase (Real-time database for the multiplayer features).

  • Deployment: Fly.io (To get it live on the web).
